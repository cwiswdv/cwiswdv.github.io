Mastering Loops: Java vs. C#

A detailed look into `while`, `for`, and `do-while` looping constructs in Java, with a quick comparison to C#.

Java Looping Syntax

Java provides three fundamental looping constructs to repeatedly execute a block of code based on a condition or for a specified number of times.

1. `while` loop

The while loop executes a block of code as long as a specified condition is true. It checks the condition before each iteration.

Syntax:

while (condition) {
    // code to be executed
}

Example:

int count = 0;
while (count < 5) {
    System.out.println("Count is: " + count);
    count++;
}

Output for the example:
Count is: 0
Count is: 1
Count is: 2
Count is: 3
Count is: 4

2. `for` loop

The for loop is ideal when you know the number of iterations in advance. It combines initialization, condition checking, and iteration updates into a single line.

Syntax:

for (initialization; condition; increment/decrement) {
    // code to be executed
}

Example:

for (int i = 0; i < 5; i++) {
    System.out.println("i is: " + i);
}

Output for the example:
i is: 0
i is: 1
i is: 2
i is: 3
i is: 4

Enhanced `for` loop (for-each):

The enhanced for loop simplifies iterating over arrays and collections.

Syntax:

for (dataType element : collectionOrArray) {
    // code to be executed for each element
}

Example:

String[] names = {"Alice", "Bob", "Charlie"};
for (String name : names) {
    System.out.println("Name: " + name);
}

Output for the example:
Name: Alice
Name: Bob
Name: Charlie

3. `do-while` loop

The do-while loop is similar to the while loop, but it guarantees that the code block will execute at least once before the condition is checked.

Syntax:

do {
    // code to be executed
} while (condition);

Note: Don't forget the semicolon ; after the while (condition) part.

Example:

int num = 0;
do {
    System.out.println("Num is: " + num);
    num++;
} while (num < 3);

// Example demonstrating at least one execution:
int x = 10;
do {
    System.out.println("This prints once even if condition is false initially. x is: " + x);
    x++;
} while (x < 10);

Output for the example:
Num is: 0
Num is: 1
Num is: 2
This prints once even if condition is false initially. x is: 10

Java vs. C# Looping Syntax: A Comparison

Java and C# are both C-style languages, and as such, their syntax for control flow structures, including loops, is strikingly similar.

Loop Type Java Syntax C# Syntax Difference
while loop 
while (condition) { /* ... */ }
 
while (condition) { /* ... */ }
 No difference.
for loop 
for (init; cond; update) { /* ... */ }
 
for (init; cond; update) { /* ... */ }
 No difference.
Enhanced for / foreach 
for (Type item : collection) { /* ... */ }
 
foreach (Type item in collection) { /* ... */ }
 Java uses :, C# uses in.
do-while loop 
do { /* ... */ } while (condition);
 
do { /* ... */ } while (condition);
 No difference.

As you can see, the core syntax for while, for, and do-while loops is virtually identical between Java and C#. The most notable difference is the keyword used in their respective enhanced/foreach loops. If you're familiar with one, you'll feel right at home with the other.
